1. 初始化与环境检查 (Setup Wizard)

首次启动向导，确保 Camoufox CLI、Unified API 和 Browser Service 正常运行。提供浏览器路径配置和账户登录。

  • 检查 Camoufox CLI 可用性
  • 验证服务端口 7701 (Unified API) 和 7704 (Browser Service)
  • 配置默认浏览器路径
  • 引导用户登录至少一个账户

2. 账户管理 (Account Management)

管理 WebAuto 使用的登录账户池，支持多平台账户的添加、删除、状态查看及重新登录。

  • 查看所有账户列表及当前登录状态
  • 添加新账户或编辑现有账户信息
  • 一键重新登录或更新账户别名

3. 任务配置 (Task Configuration)

设置爬取任务的目标、选项。支持配置导入导出，记住上次设置。

  • 设定爬取关键字、目标数量
  • 配置是否爬取正文和评论、点赞策略
  • 导入/导出任务配置，快速复用

4. 实时看板 (Dashboard)

实时监控任务执行进度、显示关键指标和日志输出。

  • 展示当前任务阶段和正在执行的操作
  • 统计成功/失败数量、耗时
  • 实时日志输出与筛选
